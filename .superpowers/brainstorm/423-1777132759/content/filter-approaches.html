Filtros: duas decisões de design

As duas features andam juntas. Preciso entender o nível de cada uma antes de propor o design completo.

1 · Chips de filtros ativos — onde ficam?

Clique na opção que parece mais natural.

Vencido × Paquímetro × Limpar tudo

A — Chips abaixo da toolbar

Aparecem entre a barra de busca/ações e a tabela. Sempre visíveis quando há filtros ativos, independente do painel de filtros estar aberto.

Filtros ●2
Vencido × Paquímetro ×

B — Badge no botão + chips dentro do painel

O botão "Filtros" ganha um badge com a contagem (●2). Os chips aparecem dentro do painel de filtros quando está aberto.

2 · URL persistence — qual escopo?

A

Todos os 4 filtros na URL

/instrumentos?status=danger&category=Paquímetro&setor=3 — navegar para detalhe e voltar restaura tudo. URL é compartilhável.

B

Só status na URL (já existe parcialmente)

Mantém o comportamento atual do status + adiciona apenas os chips sem URL sync nos demais filtros.
